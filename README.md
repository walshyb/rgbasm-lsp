# RGBASM LSP Server

A very rough proof of concept to create an LSP server (and VSCode client) for [RGBDS's](https://rgbds.gbdev.io/) [rgbasm](https://rgbds.gbdev.io/docs/v1.0.0/rgbasm.5). This is built off of VS Code's [lsp server example](https://github.com/microsoft/vscode-extension-samples/tree/main#).
