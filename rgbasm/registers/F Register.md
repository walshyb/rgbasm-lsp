- F (for flags) register is for processor statuses
- cannot be written to (save for 1 exception)
	- what is this exception?
- 1 byte large
	- 4 bits dedicated to indicate, generally, status of last operation performed
	- 1 bit dedicated to if last operation returned 0 or not
	- 1 bit dedicated to if last instruction generated a carry
	- 1 bit dedicated to if last instruction performed a subtract
	- 1 bit dedicated to a half carry indicator in case a carry is generated between nybbles in a byte
